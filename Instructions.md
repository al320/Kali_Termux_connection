# Installation Instructions

1. Clone this repository to your Kali machine:
   ```bash
    git clone https://github.com/al320/Linux-2-Termux.git
2. Go to the directory.
   ```bash
    cd Linux-2-Termux
3. Make the script executable.
   ```bash
    chmod +x cut.sh
4. Execute the script
   ```bash
   ./cut.sh
